# Font della carta vini

Carica qui i file del font **Recoleta** (in `.otf` o `.woff2`), con questi nomi esatti:

- `Recoleta-Regular.otf` — peso normale (**il più importante**, usato per i nomi dei vini)
- `Recoleta-Medium.otf` — opzionale (peso medio)
- `Recoleta-SemiBold.otf` — opzionale (titoli delle categorie)

Se hai un solo file, rinominalo in `Recoleta-Regular.otf`: verrà usato ovunque.

La carta (`carta_vini.html`) rileva questi file da sola. Finché non ci sono,
usa in automatico il font di ripiego **Fraunces**.
